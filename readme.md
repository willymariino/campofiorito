# Campo Fiorito
[![Live Demo](https://img.shields.io/badge/live-demo-brightgreen)](https://campo-fiorito.netlify.app/)

## Project Overview
Campo Fiorito is a small interactive web game created to practice core web fundamentals: HTML, CSS and JavaScript. The app renders a 10x10 grid of clickable cells, programmatically places mines, and manages game state (score and game over) through DOM manipulation and CSS state classes.

## Features

- 10x10 grid with 100 clickable cells
- 16 randomly placed mines generated on load
- Click to reveal cells; safe clicks increment a visible score
- Revealing a mine displays a bomb and triggers game over
- Visual cell states using CSS classes: closed, open, mine
- New Game button UI present to reset the session 

## Tech Stack

- **HTML5** – Semantic structure and layout  
- **CSS3** – Grid layout, visual states and responsive styling  
- **JavaScript (ES6)** – Game logic, randomization, and DOM events  
- **Netlify** – Deployment and live demo
